# PQSensing.com

Source code for [www.pqsensing.com](https://www.pqsensing.com), hosted on Namecheap.

## Repository layout

```
site/          # Live site files (HTML, CSS, JS, images)
CHANGELOG.md   # All changes since initial import
```

## Local development

Serve the site locally with Python's built-in HTTP server:

```bash
cd site
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.
